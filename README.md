# DataProcessing

Репозиторий содержит исходные таблицы и программный код для сбора, очистки и первичной обработки данных, использованных в выпускной квалификационной работе.

Тема работы:

**«Анализ мотивации проживания на сельских территориях методами машинного обучения»**

## Описание

Данный репозиторий относится к этапу подготовки данных для исследования сельских территорий.

В нем размещены таблицы с исходными и подготовленными данными, а также Python-скрипты, использованные для очистки, разведочного анализа и отбора факторов. Эти материалы применяются как основа для последующего анализа условий проживания, инфраструктурной обеспеченности, пространственной доступности и миграционных процессов.

## Состав репозитория

`Answers.docx` — файл с визуальными материалами по результатам социологического опроса.

`clean.py` — скрипт первичной очистки и подготовки статистических таблиц.

`analiz.py` — скрипт разведочного анализа данных. Используется для построения графиков распределения численности населения, дорожных расстояний и количества инфраструктурных объектов.

`signs.py` — скрипт для анализа факторов, связанных с миграционными процессами. Используется для корреляционного анализа и оценки важности признаков.

`village.xlsx` — таблица с показателями условий жизни сельского населения.

`villagedem.xlsx` — таблица с миграционными показателями сельского населения.

`population_NN.xlsx`, `results_NN.xlsx`, `distances_matrix_NN.xlsx` — данные по Нижегородской области: численность населения территорий, инфраструктурные объекты и матрица дорожных расстояний.

`population_kirov.xlsx`, `results_kirov.xlsx`, `distances_matrix_kirov.xlsx` — данные по Кировской области: численность населения территорий, инфраструктурные объекты и матрица дорожных расстояний.

## Использование данных

Данные из данного репозитория используются для:

1. анализа условий проживания на сельских территориях;
2. выявления факторов, связанных с миграционными процессами;
3. формирования признакового пространства;
4. анализа инфраструктурной обеспеченности территорий;
5. оценки пространственной доступности;
6. проверки применимости подхода на данных Кировской области.

## Основные этапы обработки

В рамках подготовки данных выполнялись следующие действия:

1. сбор открытых статистических и инфраструктурных данных;
2. очистка таблиц и приведение данных к единому виду;
3. проверка структуры таблиц и типов данных;
4. сопоставление территорий между разными источниками;
5. расчет и анализ дорожных расстояний;
6. разведочный анализ данных;
7. отбор факторов, связанных с условиями проживания и миграционными процессами.

## Назначение репозитория

Репозиторий используется как открытое хранилище исходных данных, промежуточных таблиц и скриптов обработки, применённых при выполнении выпускной квалификационной работы.

Материалы из данного репозитория позволяют проследить этап подготовки данных: от первичных таблиц и результатов опроса до формирования наборов данных, использованных для дальнейшего анализа сельских территорий.
